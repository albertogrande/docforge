# Core concepts

Forge models documentation as **structured, provenance-tracked, human-gated** content.

## Lifecycle

A page moves `stub -> draft -> reviewed`. An agent may only reach `draft`; promotion to
`reviewed` requires a human approving the pull request. See [Getting Started](getting-started.md).
